Exercice Docker

Introduction

Docker est une plateforme permettant de développer, distribuer et exécuter des applications dans des conteneurs. Cet exercice vous aidera à créer et exécuter votre premier conteneur Docker.

Étapes de l'exercice

1. Créer un Dockerfile

Créez un fichier nommé Dockerfile avec le contenu suivant :

FROM nginx:alpine
COPY index.html /usr/share/nginx/html/
EXPOSE 80

2. Créer index.html

Créez un fichier nommé index.html avec ce contenu :

<!DOCTYPE html>
<html>
<head>
    <title>Test Docker</title>
</head>
<body>
    <h1>Bonjour depuis Docker !</h1>
    <p>Si vous voyez ceci, votre conteneur Docker fonctionne.</p>
</body>
</html>

3. Construire l'image Docker

Exécutez cette commande pour construire votre image Docker :

docker build -t mon-app-docker .

4. Lancer le conteneur

Démarrez votre conteneur avec :

docker run -d -p 8080:80 mon-app-docker

Accédez à votre application sur http://localhost:8080

Commandes Docker Essentielles

  • docker images Lister toutes les images Docker
  • docker ps Lister les conteneurs en cours d'exécution
  • docker stop [id-conteneur] Arrêter un conteneur en cours d'exécution
  • docker rm [id-conteneur] Supprimer un conteneur
  • docker rmi [id-image] Supprimer une image
